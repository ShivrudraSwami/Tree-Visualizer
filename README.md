## üöÄ Key Features (Resume Highlights)
- **Interactive Visualization:** Visualizes 5+ complex algorithms including BFS, DFS, ZigZag, Boundary, and Vertical Traversal.
- **Optimized Performance:** Utilizes **`React.memo`** to minimize re-renders, ensuring smooth 60fps animations even with 100+ nodes.
- **Real-Time Animation:** Step-by-step path tracing with dynamic color coding for active and visited nodes.
- **Backend Logic:** Python Flask backend implements algorithms mimicking **C++ STL logic** (using `collections.deque` and `defaultdict`) for interview-standard accuracy.
